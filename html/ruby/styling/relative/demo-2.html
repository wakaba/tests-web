反復ブロック (HTML)

[1] repeat 属性値が整数である要素を、 反復(はんぷく/repetition)ブロック(block)といいます。 [WF2 3.2.2]

[2] 仕様書:

  • Web Forms 2.0
    • 3.2.2. Repetition blocks Web Forms 2.0 (WD)

構文

[3] repeat 属性

反復ブロックの repeat 属性値は、 整数です。すなわち、 省略可能な - の後に1つ以上の十進数の数字が続く文字列です。 [WF2 3.2.2]

[4] repeat-template 属性

反復ブロックは、 repeat-template 属性を使って、関連付ける反復雛形を指定して構いません。 [WF2 3.2.2]


( [] )

フォーム

[5] 反復雛形

  • [6] 反復ブロックに repeat-template 属性があり、その属性値と等しい識別子を持つ要素がある場合、
    • [7] その要素が反復雛形なら、それが、 その反復ブロックが関連付けられる(associated)反復雛形です。 なお、同じ識別子の要素がある場合の動作は、 getElementById と同じとするべきです。 [WF2 3.2.2]
    • [8] その要素が反復雛形でなければ、 その反復ブロックはどの反復雛形とも関連付けられません。 [WF2 3.2.2]
  • [9] 反復ブロックに repeat-template 属性がない場合は、 反復雛形である最初の弟があるなら、それと関連付けられます。 [WF2 3.2.2]

一般に反復雛形は反復ブロックの弟である必要はありませんが、 反復雛形の repetitionBlocks には現れないことになります。 [WF2 3.2.2]

[10] 孤児反復ブロック

関連付けられた反復雛形のない反復ブロックを、 孤児(こじ/orphan)反復(はんぷく/repetition)ブロック(block)といいます。 孤児反復ブロックは削除されたり移動されたりすることはありますが、 追加の影響は受けません。 [WF2 3.2.2]

[11] 索引

反復ブロックは索引(さくいん/index)を持ちます。 索引の初期値は、 repeat 属性値です。 [WF2 3.2.2]


( [] )

歴史

[13] 反復ブロックは、 Web Forms 2.0 で HTML に導入されました。


( [] )

例

[12] 

<div>
 <div repeat="0"/> <!-- A simple repetition block, index 0. -->
 <div repeat="-5"/> <!-- Another, index -5 -->
 <div repeat="2"/> <!-- A simple repetition block, index 2. -->
 <div repeat="nothing"/> <!-- Just a normal element. -->
 <div repeat=" 3"/> <!-- Another normal element (leading whitespace). -->
 <div repeat="template"/> <!-- The template for the last few elements. -->
 <div repeat="1"/> <!-- Orphan repetition block, index 1. -->
</div>
<div repeat="0"/> <!-- Orphan repetition block, index 0. -->

[WF2 3.2.2]
